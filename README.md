# react-native-one-armed-bandit

## 需要环境

React-Native

IOS Xcode >= 7.0
Android Android JDK
##安装教程
[React-Native中文网](http://http://reactnative.cn/docs/0.39/getting-started.html)

## 运行测试

### IOS

```
$ react-native run-ios
```
### Android
```
$ react-native run-android
```
